2. Before you pack

  1. Close the Thuraya app so the SQLite database is not being edited while the archive is built.
  2. Make sure the project is in its canonical OneDrive folder.
  3. If you recently changed anything, launch the app once and confirm the latest data is visible before packing.
  4. If you want a quick safety check first, use Run Dry-Run Pack.

3. What happens during pack

  1. The tool scans the project tree and excludes only temporary cache/archive folders.
  2. It calculates a SHA-256 manifest for each included file.
  3. It writes a single .tar.gz archive into G:\My Drive\PG_Backups\codex_thuraya_rcs.
  4. That archive can then be copied to USB, network share, cloud storage, or any other transfer method.

4. How to use the result

  1. After pack finishes, open G:\My Drive\PG_Backups\codex_thuraya_rcs.
  2. Copy the newest thuraya_prepay_airtime_YYYYMMDDThhmmss.tar.gz file to the destination computer.
  3. On the destination computer, open this same project kit and run Run Thuraya Unpack.cmd.
