1. What you need on the new computer

ItemWhy
Python 3The Windows launchers and the unpack UI run with Python.
The Thuraya portable archiveThis contains the project, database, dependencies, and assets.
This portable kit folderIt provides the unpack script, HTML guide, and launchers.
Because the app uses SQLite, there is no PostgreSQL install, database server setup, or service restore step here. The database is just the file data/thuraya_airtime.sqlite3.
The unpack screen reads the portable archives from G:\My Drive\PG_Backups\codex_thuraya_rcs by default.

2. Recommended target path

For the closest match to your current setup, restore into:

C:\Users\<YourUser>\OneDrive\Documents\Codex\scripts

The unpack process will then recreate the project folder underneath that root as Thuraya Prepay Airtime.

3. Safe unpack order

  1. Use Run Dry-Run Unpack first to confirm the selected archive and the destination path.
  2. Run the full unpack.
  3. Open Launch Thuraya Prepay Airtime Sales.cmd in the restored folder.
  4. If Python is installed but a dependency is somehow missing, run Install Dependencies.cmd once. Normally the bundled vendor folder should already be enough.

4. What the restore does

  1. Reads the selected archive and its metadata.
  2. Extracts it to a temporary staging folder.
  3. Verifies each file against the SHA-256 manifest.
  4. Copies the verified project tree into the chosen target root.
  5. Leaves you with a runnable Windows folder matching the packed source machine.
Old sales Excel/PDF files stored in Downloads are still outside the app folder. If you need those on the new machine, copy them separately after the app is restored.
